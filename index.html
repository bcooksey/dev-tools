Dev Tools

Welcome! Below is a quick summary of the available tools.

  • /converters - The main page that has lots of widgets for doing data manipulation like pretty printing and timestamp conversions
  • /list - Endpoint that lists out the files this server can serve. Use `?type=<extension>` to filter list down to a specific filetype
  • /data/<some_file> - Endpoint that will serve up the file specified by <some_file> path part. Add a `octet=true` to stream the file via application/octet-stream.
  • /upload - You can POST form-encoded data here to experiment with uploading files to a server
